 Для корректной работы MSSPEECH нужно:

 --------------------------------------

 1) Установить MicrosoftSpeechPlatformSDK.msi 
 2) Установить SpeechPlatformRuntime.msi (и/или SpeechPlatformRuntime_x64.msi для поддержки x64 архитектуры приложения)
 3) Установить MSSpeech_SR_ru-RU_TELE.msi (для поддержки SR для RU/UA региона)
 4) Закинуть папку Grammars, в root деректорию проекта библиотеки

 Дополнительно:
 1) Для показа панели дебага переустановить ZedGraph
 2) Если есть проблемы с возращением COM "библиотека не заррегистрирована" переустановить  пункты 1,2,3 установить DetectorAMD как ехе приложение 32 bit
